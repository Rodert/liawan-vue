## 环境搭建


**启动前端：**

1. 依赖环境：

- npm
- nodejs

2. 加载依赖：

```
npm install --registry=https://registry.npm.taobao.org
```

3. 启动：

```
npm run dev
```

**启动后端：**

1. 依赖环境：

- jdk8（及以上）
- mysql（建议v5.7）
- redis

2. 启动

执行：main

```
com/javapub/liawan/admin/LiaWanApplication.java
```
